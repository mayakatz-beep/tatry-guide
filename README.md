# Tatry Guide

מדריך טיול משפחתי אינטראקטיבי לסלובקיה, מוכן לפרסום ב-GitHub Pages.

## שימוש

1. חלצי את קובץ ה-ZIP.
2. העלי את `index.html` לשורש הריפו `tatry-guide`.
3. ב-GitHub עברי ל-Settings → Pages.
4. ודאי שהפרסום מוגדר מ-`main` ומתיקיית השורש.

האתר כולל מסלול יומי, רשימות הזמנות ואריזה, מצב לילה, שיתוף והדפסה.
